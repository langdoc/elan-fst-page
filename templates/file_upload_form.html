ELAN annotation pipeline

This online version of the annotation tool is developed by Niko Partanen, and it runs in CSC's cPouta infrastructure. The tool is based to collaboration and contributions of Ciprian Gerstenberger, Niko Partanen, Mika Hämäläinen, Iris Perkmann, Michael Rießler and Joshua Wilbur. Further documentation of the pipeline is available here.

This annotation tool works with ELAN files that follow the standards used in projects associated with the Freiburg Research Group in Saami Studies, and in Giellagas Institute for Saami Studies in the University of Oulu.

The files are first validated, and the files have to follow one of those structures. Customizing the tool according to a new template is also possible.

Current implementation supports tier template used in Komi documentation project IKDP, and the template used in Giellagas Corpus of Spoken Saami Languages. This work has been carried out in Kone Foundation funded project Language Documentation Meets Language Technology: The Next Step in the Description of Komi
