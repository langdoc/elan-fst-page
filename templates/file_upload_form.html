ELAN annotation pipeline

This online version of the annotation tool is developed by Niko Partanen, and it runs in CSC's cPouta infrastructure. The tool is based to collaboration and contributions of Ciprian Gerstenberger, Niko Partanen, Mika Hämäläinen, Iris Perkmann, Michael Rießler and Joshua Wilbur. Further documentation of the pipeline is available here.

Current implementation supports tier template used in Komi documentation project IKDP, and the template used in Giellagas Corpus of Spoken Saami Languages. Extending it into new templates and languages is always possible. This work has been carried out in Kone Foundation funded project Language Documentation Meets Language Technology: The Next Step in the Description of Komi

Known bugs and upcoming improvements

  • Language should be selectable from the languages present in GiellaLT, or read directly from the tier information
  • Compounds get broken now in an unintuitive way
  • Some quotation marks are not processed correctly now.

Fixed bugs

  • Empty utterances are handled
  • Multiple transcription tiers are handled, but one has to be specified as the primary transcription that is tokenized and annotated. This must be clear on tier template level.
